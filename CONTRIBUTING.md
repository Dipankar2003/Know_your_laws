How to contribute ?

Research data you want to contribute like laws and regulations, articles of constitution.
Copy the example from below and fill your research in respected coloum in json file.
Recheck your code and push request. (wrong syntax code can be rejected)
You done with your contribution.

sample:

{
        "id": 1,

        "introduced-year": "1950",
